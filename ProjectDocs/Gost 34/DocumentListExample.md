# Пример перечня документов

Для большого проекта внедрения сложной системы может потребоваться достаточно большой перечень документации. Приведем пример такого перечня в таблице ниже.  
  

| Этап                                                            | Документ                                                                                                                                                                                                                                                                                                                                                                                                                                                                            | Содержание документа                                                                                                                                                                                                                      | Примечания |
| --------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- |
| 1. Техническое проектирование                                   | Ведомость технического проекта                                                                                                                                                                                                                                                                                                                                                                                                                                                      | Перечень документов технического проекта                                                                                                                                                                                                  |            |
| Пояснительная записка к техническому проекту                    | — описание основных технических решений;  <br>— описание процесса деятельности с применением системы;  <br>— мероприятия по подготовке объекта автоматизации к вводу системы в действие                                                                                                                                                                                                                                                                                             | При поставке типовых продуктов не разрабатывается                                                                                                                                                                                         |            |
| Описание автоматизируемых функций                               | — детальные сценарии работы системы;  <br>— макеты пользовательского интерфейса с подробным описанием элементов;  <br>— перечень подсистем (модулей) с описанием выполняемых функций;  <br>— описание структуры данных, где это необходимо;  <br>— формы и правила формирования отчетов                                                                                                                                                                                             | При модернизации приводится описание модернизируемых функций в формате «как должно быть».  <br>— При поставке типовых продуктов не разрабатывается                                                                                        |            |

| Описание организационной структуры                              | Изменения в организационной структуре, необходимые для функционирования системы                                                                                                                                                                                                                                                                                                                                                                                                     | При модернизации разрабатывается в случае внесения изменений в оргструктуру                                                                                                                                                               |            |
| Смета                                                           | Уточненная стоимость работ по созданию и внедрению системы                                                                                                                                                                                                                                                                                                                                                                                                                          |                                                                                                                                                                                                                                           |            |
